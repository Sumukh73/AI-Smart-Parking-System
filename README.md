# 🚗 AI Smart Parking System
> AI-powered smart parking system using computer vision for real-time slot detection, allocation, and optimization.
---
## 📌 Overview
This project presents an intelligent parking solution that helps users find available parking spaces in real time. It uses AI and computer vision techniques to detect vehicles and optimize parking allocation, reducing traffic congestion and improving urban mobility.
---
## 🎯 Key Features
- 🔍 Real-time parking slot detection  
- 🤖 AI-based vehicle detection (YOLOv8)  
- 📍 Smart parking slot recommendation  
- ⏱️ Reduced search time and fuel wastage  
- 🌆 Scalable for smart city integration  
---
## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Computer Vision:** OpenCV  
- **AI Model:** YOLOv8 (Ultralytics)  
- **Backend:** Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** (Optional – MongoDB / PostgreSQL)

## 🔄 Workflow
1. User searches for parking location  
2. System collects real-time data (CCTV / video input)  
3. AI model detects available parking slots  
4. System recommends optimal parking  
5. User navigates and parks
